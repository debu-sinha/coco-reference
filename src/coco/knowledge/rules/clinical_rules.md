# Clinical Domain Knowledge

## Condition-Drug Mappings

### Type 2 Diabetes Mellitus (T2DM)
- **ICD-10 Codes**: E11.x series
- **First-Line Medications**: Metformin (Biguanides)
- **Second-Line Options**: 
  - Sulfonylureas (e.g., glipizide, glyburide)
  - GLP-1 Receptor Agonists (e.g., semaglutide, dulaglutide)
  - SGLT-2 Inhibitors (e.g., empagliflozin, canagliflozin)
  - DPP-4 Inhibitors (e.g., sitagliptin, saxagliptin)
- **Common Combinations**: Metformin + GLP-1 RA, Metformin + SGLT-2i
- **Insulin**: Reserved for advanced disease or poor control despite oral agents
- **Monitoring**: HbA1c tests typically every 3-6 months
- **Clinical Goals**: HbA1c < 7% (individualized), LDL < 100 mg/dL

### Hypertension (HTN)
- **ICD-10 Codes**: I10, I11, I12, I13 (primary, secondary, with comorbidities)
- **First-Line Medications**:
  - ACE Inhibitors (e.g., lisinopril, enalapril)
  - Angiotensin II Receptor Blockers (ARBs) (e.g., losartan, valsartan)
  - Calcium Channel Blockers (e.g., amlodipine, diltiazem)
  - Thiazide Diuretics (e.g., hydrochlorothiazide)
- **Special Considerations**:
  - ACE-I or ARB in diabetic patients with HTN (renal protection)
  - Beta-blockers for post-MI patients with HTN
  - Avoid NSAIDs (elevate BP, reduce drug efficacy)
- **Monitoring**: Blood pressure checks at visits, home BP monitoring
- **Clinical Goals**: < 130/80 mmHg for most patients, < 140/90 for older adults

### Coronary Artery Disease (CAD) / Acute Coronary Syndrome (ACS)
- **ICD-10 Codes**: I21.x (MI), I24.x (complications), I25.x (chronic ischemic heart disease)
- **Key Medications**:
  - Antiplatelet: Aspirin, clopidogrel (Plavix), ticagrelor
  - Beta-blockers: Metoprolol, carvedilol, bisoprolol
  - ACE-I/ARB: Lisinopril, ramipril, losartan
  - Statins: High-intensity preferred (atorvastatin 80mg, rosuvastatin 20-40mg)
  - Nitrates: For angina symptoms
- **Procedures**: Angiography (CPT 93510), Percutaneous Coronary Intervention (PCI) (CPT 92920-92944), Coronary Artery Bypass Graft (CABG) (CPT 33510-33548)
- **Monitoring**: EKG (CPT 93000), Stress testing, Echocardiography
- **Post-MI**: Dual antiplatelet therapy (DAPT) for 12 months typical

### Heart Failure (HF)
- **ICD-10 Codes**: I50.x series (HF with reduced ejection fraction I50.2x, preserved I50.3x)
- **Systolic HF (HFrEF) Medications**:
  - ACE-I/ARB as foundational (lisinopril, enalapril, losartan)
  - Beta-blocers: Carvedilol, metoprolol succinate
  - Aldosterone antagonist: Spironolactone, eplerenone
  - SGLT-2 inhibitors: Dapagliflozin, empagliflozin (newer guideline standard)
- **Diastolic HF (HFpEF)**: Focus on rate control, diuretics; limited pharmacotherapy options
- **Diuretics**: Furosemide, spironolactone for fluid management
- **Monitoring**: Echocardiography (ejection fraction), NT-proBNP/BNP levels
- **Procedures**: Device therapy for reduced EF: Pacemakers (CPT 33206-33249), ICDs (CPT 33240-33280)

### Chronic Obstructive Pulmonary Disease (COPD)
- **ICD-10 Codes**: J43.x, J44.x
- **Maintenance Medications**:
  - Inhaled corticosteroids (ICS): Fluticasone, budesonide
  - Long-acting beta-2 agonists (LABA): Salmeterol, formoterol
  - Long-acting muscarinic antagonists (LAMA): Tiotropium, umeclidinium
  - Combination inhalers: Most patients use ICS/LABA or ICS/LABA/LAMA
- **Acute Exacerbation**: Systemic corticosteroids, antibiotics if bacterial
- **Monitoring**: Spirometry (lung function tests), Chest X-ray
- **Procedures**: Pulmonary function testing, Bronchodilator challenge

### Atrial Fibrillation (AFib)
- **ICD-10 Codes**: I48.x
- **Anticoagulation**: Critical for stroke prevention
  - Warfarin (requires INR monitoring)
  - DOACs (Dabigatran, rivaroxaban, apixaban, edoxaban)
  - CHADS2-VASc score determines need for anticoagulation
- **Rate Control**:
  - Beta-blockers: Metoprolol, atenolol
  - Calcium channel blockers: Diltiazem, verapamil
  - Digoxin: Older option
- **Rhythm Control**: Antiarrhythmics (amiodarone, flecainide) for select patients
- **Procedures**: Ablation (CPT 93650-93660), Cardioversion (CPT 92960-92961)
- **Monitoring**: EKG (CPT 93000), Holter monitoring, Echocardiography

### Asthma
- **ICD-10 Codes**: J45.x
- **Controller Medications**:
  - ICS (inhaled corticosteroids): Fluticasone, budesonide (first-line)
  - ICS/LABA combinations: For moderate-to-severe asthma
  - LTRA (leukotriene receptor antagonists): Montelukast
  - Biologic agents: Omalizumab (IgE-mediated), Dupilumab (allergic/eosinophilic)
- **Rescue Medications**: Short-acting beta-2 agonists (SABA): Albuterol/salbutamol
- **Monitoring**: Peak flow monitoring, Spirometry (CPT 94010-94070)
- **Procedures**: Allergy testing, Immunotherapy

### Dyslipidemia
- **ICD-10 Codes**: E78.x
- **Primary Medications**: Statins (high-intensity standard)
  - High-intensity: Atorvastatin 40-80mg, Rosuvastatin 20-40mg
  - Moderate-intensity: Atorvastatin 10-20mg, Rosuvastatin 5-10mg, Simvastatin 20-40mg
- **Add-on Therapies**:
  - Ezetimibe: For additional LDL reduction
  - PCSK9 inhibitors: For very high-risk patients or statin-intolerant
  - Bempedoic acid: Newer agent
- **Monitoring**: Lipid panel (CPT 80061, 82465) every 4-12 weeks initially, then annually
- **Clinical Goals**: LDL < 70 for CAD patients, < 100 for others

### Depression
- **ICD-10 Codes**: F32.x (major depressive disorder), F33.x (recurrent)
- **First-Line Medications**:
  - SSRIs: Sertraline, escitalopram, paroxetine, fluoxetine
  - SNRIs: Venlafaxine, duloxetine
  - Bupropion: Atypical, good for anhedonia and fatigue
- **Second-Line**: Tricyclic antidepressants, MAOIs (if TCA/SNRI/SSRI failed)
- **Augmentation**: Addition of second agent when monotherapy insufficient
- **Monitoring**: Behavioral health visits, Symptom rating scales
- **Procedures**: Psychotherapy, ECT (electroconvulsive therapy) for severe cases
- **Cautions**: SEROTONIN SYNDROME risk with combination therapy

### Anxiety Disorders
- **ICD-10 Codes**: F41.1 (generalized anxiety), F40.x (phobic), F43.10 (PTSD)
- **First-Line Medications**:
  - SSRIs: Sertraline, paroxetine, escitalopram (first-line for chronic)
  - SNRIs: Venlafaxine, duloxetine
- **Benzodiazepines**: Short-term only, risk of dependence (alprazolam, lorazepam)
- **Other**: Buspirone (anxiolytic, less abuse potential)
- **Monitoring**: Psychiatry visits, Function assessments

### Chronic Kidney Disease (CKD)
- **ICD-10 Codes**: N18.x (Stages 1-5), N19 (ESRD)
- **Key Medications**:
  - ACE-I/ARB: Cornerstone of renal protection (lisinopril, losartan)
  - SGLT-2 inhibitors: Emerging evidence for renal protection (empagliflozin, dapagliflozin)
  - Blood pressure control: Amlodipine, hydrochlorothiazide
  - Phosphate binders (Stage 3b+): Calcium acetate, sevelamer
  - ESA (Erythropoiesis Stimulating Agents) for anemia if GFR < 30
- **Monitoring**: Serum creatinine, eGFR, Urine albumin-to-creatinine ratio (UACR)
- **Procedures**: Renal ultrasound, Renal biopsy if diagnosis unclear
- **Referral**: Nephrology at Stage 3b or earlier if rapidly declining

## ICD-10 Code Hierarchies

### Hierarchical Structure
ICD-10-CM codes follow a hierarchical structure from general to specific:

```
Category (1 letter + 2 digits): E11 (Type 2 DM)
|-  Subcategory (. + 1 char): E11.2 (With renal complications)
|-  Deeper specificity (+ 2 chars): E11.21 (With diabetic nephropathy)
\-  Most specific: E11.21 (With diabetic nephropathy without chronic kidney disease)
```

### Diabetes (E10-E14)
- E10: Type 1 Diabetes
- E11: Type 2 Diabetes
- E12: Malnutrition-Related Diabetes
- E13: Other Specified Diabetes
- E14: Unspecified Diabetes
- **Second Character** (complication):
  - E11.0: Uncomplicated
  - E11.1: With ketoacidosis
  - E11.2: With hyperosmolarity
  - E11.6: With hypoglycemia
  - E11.9: With unspecified complications

### Cardiovascular (I10-I50)
- I10-I15: Hypertensive diseases
- I20-I25: Ischemic heart diseases (CAD, MI, angina)
- I30-I52: Other forms of heart disease (HF, myocarditis, pericarditis)
- I60-I69: Cerebrovascular diseases (stroke, TIA)
- I70-I79: Diseases of arteries, arterioles, capillaries
- I80-I89: Diseases of veins (DVT, thrombophlebitis)

### Respiratory (J00-J99)
- J40-J47: Chronic lower respiratory diseases (COPD, asthma, bronchiectasis)
- J84: Interstitial lung diseases (pulmonary fibrosis)
- J85-J86: Suppurative and necrotic conditions
- J90-J99: Other respiratory diseases

## Drug Class Relationships

### Antihypertensive Mechanism Classes

**ACE Inhibitors** (-pril)
- Mechanism: Block angiotensin II formation
- Examples: Lisinopril, enalapril, ramipril, perindopril
- Side Effect: Persistent dry cough (10-20%)
- Caution: Hyperkalemia risk, contraindicated in pregnancy

**Angiotensin Receptor Blockers** (-sartan)
- Mechanism: Block angiotensin II type 1 receptor
- Examples: Losartan, valsartan, olmesartan, candesartan
- Advantage: No cough (unlike ACE-I)
- Similar efficacy to ACE-I, often second choice if ACE-I intolerant

**Calcium Channel Blockers** (-dipine for dihydropyridines)
- Mechanism: Block L-type calcium channels, reduce vascular tone
- Dihydropyridines (vasodilatory): Amlodipine, nifedipine (swelling)
- Non-dihydropyridines (rate-limiting): Diltiazem, verapamil (slow HR)
- Good for isolated systolic hypertension in elderly

**Thiazide and Thiazide-Like Diuretics**
- Mechanism: Block NaCl reabsorption in distal convoluted tubule
- Examples: Hydrochlorothiazide (HCTZ), chlorthalidone
- Side Effects: Hypokalemia, hyperglycemia, hyperuricemia
- Often used in combination therapy

**Beta-Blockers** (-olol)
- Mechanism: Block beta-1 and beta-2 adrenergic receptors
- Examples: Metoprolol, atenolol, carvedilol, bisoprolol
- Special Use: Post-MI, HF, arrhythmias
- Drawback: Fatigue, sexual dysfunction, contraindicated in asthma/COPD

### Lipid-Lowering Drug Classes

**Statins** (-statin)
- Mechanism: HMG-CoA reductase inhibitors, reduce cholesterol synthesis
- High-intensity: Atorvastatin 40-80mg, Rosuvastatin 20-40mg
- Moderate-intensity: Atorvastatin 10-20mg, Pravastatin 40-80mg
- Side Effect: Muscle myopathy, elevated liver enzymes
- Most commonly prescribed class

**PCSK9 Inhibitors**
- Mechanism: Monoclonal antibodies blocking PCSK9, increase LDL receptor
- Examples: Evolocumab, alirocumab
- Use: For very high-risk patients or statin intolerance
- Administration: Injectable (every 2-4 weeks)

**Ezetimibe**
- Mechanism: Inhibits cholesterol absorption in small intestine
- Adjunctive therapy to statins for additional LDL reduction
- Modest effect (10-20% LDL reduction)

### Diabetes Medication Classes

**Biguanides**
- Mechanism: Reduce hepatic glucose production, improve insulin sensitivity
- Example: Metformin (first-line)
- Caution: Lactic acidosis risk (rare), renal impairment contraindication

**Sulfonylureas** (-sulfonylurea)
- Mechanism: Stimulate pancreatic beta-cells to release insulin
- Examples: Glipizide, glyburide, glimepiride
- Drawback: Hypoglycemia risk, weight gain
- Declining use due to newer agents

**GLP-1 Receptor Agonists** (-tide, -ade)
- Mechanism: Mimic glucagon-like peptide 1, enhance insulin secretion
- Examples: Semaglutide (Ozempic), dulaglutide (Trulicity), liraglutide (Victoza)
- Benefit: Weight loss, cardiovascular protection, once-weekly dosing
- Growing first or second-line option for T2DM

**SGLT-2 Inhibitors** (-flozin)
- Mechanism: Inhibit sodium-glucose cotransporter, increase urinary glucose
- Examples: Empagliflozin, canagliflozin, dapagliflozin
- Benefits: Cardiovascular protection, HF benefit, weight loss
- Monitor: Genital infections, euglycemic DKA (rare)

**DPP-4 Inhibitors** (-gliptin)
- Mechanism: Prevent GLP-1 degradation, prolong incretin action
- Examples: Sitagliptin, saxagliptin, linagliptin
- Neutral on weight, well-tolerated
- Second-line agent, less effective than GLP-1 RAs

## Common Cohort Inclusion/Exclusion Criteria Patterns

### Temporal Inclusion Criteria
```
1. Index Date Definition:
   - Diagnosis date (first occurrence of ICD-10 code)
   - Medication start date (first prescription fill)
   - Procedure date (first occurrence of CPT code)
   - Claims-based index (earliest claim with condition code)

2. Continuous Enrollment:
   - Minimum 12 months pre-index (baseline period)
   - Minimum 12 months post-index (follow-up period)
   - Must be enrolled at index date
   - Query: WHERE enrollment_start <= index_date 
           AND (enrollment_end IS NULL OR enrollment_end >= follow_up_end_date)

3. Age Filtering:
   - At index date: WHERE age >= 18 AND age <= 75
   - Age-specific cohorts: >= 65 (elderly), >= 18 and <= 40 (working age)
```

### Diagnosis-Based Cohorts
```
Pattern 1: Single Condition
  WHERE icd10_code = 'E11.9' 
    AND diagnosis_date BETWEEN start_date AND end_date

Pattern 2: Condition Family (ICD-10 hierarchy)
  WHERE icd10_code LIKE 'E11%'
    AND diagnosis_date BETWEEN start_date AND end_date

Pattern 3: Condition with Comorbidity
  WHERE EXISTS (
    SELECT 1 FROM diagnoses d1 
    WHERE d1.patient_id = patients.patient_id 
      AND d1.icd10_code LIKE 'E11%'
  )
  AND EXISTS (
    SELECT 1 FROM diagnoses d2 
    WHERE d2.patient_id = patients.patient_id 
      AND d2.icd10_code LIKE 'I10%'
  )

Pattern 4: Excluding Specific Condition
  WHERE icd10_code LIKE 'E11%'
    AND NOT EXISTS (
      SELECT 1 FROM diagnoses d_ex 
      WHERE d_ex.patient_id = patients.patient_id 
        AND d_ex.icd10_code LIKE 'E10%'  -- Type 1 diabetes
    )
```

### Medication-Based Cohorts
```
Pattern 1: On Specific Drug
  WHERE generic_name = 'metformin' 
    AND rx_date BETWEEN start_date AND end_date

Pattern 2: Drug Class (pattern matching)
  WHERE generic_name IN ('atorvastatin', 'rosuvastatin', 'simvastatin')
    AND rx_date BETWEEN start_date AND end_date

Pattern 3: Multiple Lines of Therapy (sequential)
  WITH ordered_rx AS (
    SELECT patient_id, generic_name, rx_date,
           LAG(generic_name) OVER (PARTITION BY patient_id ORDER BY rx_date) as prev_drug
    FROM prescriptions
    WHERE rx_date BETWEEN start_date AND end_date
  )
  WHERE generic_name IN ('sitagliptin', 'glipizide')
    AND prev_drug = 'metformin'  -- Step therapy

Pattern 4: Not on Medication (exclusion)
  WHERE NOT EXISTS (
    SELECT 1 FROM prescriptions 
    WHERE patient_id = patients.patient_id 
      AND generic_name = 'warfarin'
      AND rx_date BETWEEN start_date AND end_date
  )
```

### Exclusion Criteria Patterns
```
1. Pregnancy-Related Exclusions:
   WHERE NOT EXISTS (
     SELECT 1 FROM diagnoses 
     WHERE patient_id = patients.patient_id 
       AND icd10_code LIKE 'O%'  -- Pregnancy codes
       AND diagnosis_date BETWEEN pregnancy_start AND pregnancy_end
   )

2. Type 1 Diabetes Exclusion (for T2DM cohorts):
   WHERE NOT EXISTS (
     SELECT 1 FROM diagnoses 
     WHERE patient_id = patients.patient_id 
       AND icd10_code LIKE 'E10%'
   )

3. Active Malignancy Exclusion:
   WHERE NOT EXISTS (
     SELECT 1 FROM diagnoses 
     WHERE patient_id = patients.patient_id 
       AND icd10_code LIKE 'C%'  -- Cancer codes
       AND diagnosis_date >= (index_date - 365)
   )

4. Advanced Kidney Disease Exclusion:
   WHERE NOT EXISTS (
     SELECT 1 FROM diagnoses 
     WHERE patient_id = patients.patient_id 
       AND icd10_code LIKE 'N18.4%'  -- Stage 4 CKD
   )

5. Organ Transplant Exclusion:
   WHERE NOT EXISTS (
     SELECT 1 FROM procedures 
     WHERE patient_id = patients.patient_id 
       AND cpt_code IN ('33935', '33940', '50360', '50365', '32850')  -- Transplant codes
   )
```
